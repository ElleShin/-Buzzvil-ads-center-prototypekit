# 레퍼런스 이미지

이 폴더에 Buzzvil Ads Center 실제 화면 캡처 이미지를 넣어주세요.

## 업로드 방법

1. 이 폴더의 이미지를 모두 선택
2. Claude.ai → 프로젝트 → **Files** → `+` 버튼 → 이미지 업로드

아래 화면들을 캡처해서 넣어두면 Claude가 더 정확하게 UI를 생성할 수 있어요.

## 주의

- 파일명은 자유롭게 해도 되지만, 위 권장 이름을 따르면 관리가 편해요
- PNG 또는 JPG 형식 권장
- 해상도는 높을수록 좋아요 (Retina 캡처 권장)
